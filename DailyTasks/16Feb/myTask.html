Task:

  • Create a remote repo
  • Clone remote repo to local dir
  • Make changes to cloned repo
  • push modification to remote repo
  • fetch updated files from remote repo

Solution:

  • To clone project use 'git clone repo_url' ; fynd_academy repo is cloned from remote created by other user
  • To push changes:
    • use 'git push origin branch_name' if you are collaborator
    • create pull request if not collaborator, for this push to your public repo then create pull request
  • To get updated changes from repo:
    • use 'git fetch origin'. this will store remote commit separate to current local working tree.
      To view local branches 'git branch'. to view remote repo 'git branch -I'
    • use 'git pull origin'. This will download remote repo and auto merge to current local working tree.
