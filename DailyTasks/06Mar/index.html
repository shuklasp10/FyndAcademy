Color coding

  • hex - six digit code in hexadecimal format; #FF0000; case insensitive
  • rgb - commbination of red green blue; support value from 0 to 255
  • RGBA - rgb with alpha value; opacity of color; range from 0 (transparent) to 1(opaque)
  • hsl - hue(color)-saturation(intensity of color)-lightness(brightness of color); hsl(0,100%,50%)
  • hsla - hsl with alpha value; same as rgba; hsla(0,100%,50%,0.5)
