Git commands

  • git init: initialize empty git repo in working directory
  • git add filename : add untracked file to staging area
  • git add . : add all files to staging area
  • git restore --staged filename: to unstage files
  • git commit -m "message": commit all files from staging area with a commit message
  • git status: shows current status of working directory
  • git checkout filename: revert changes made to file if not commited yet
  • git checkout . : revert all files before commit
  • git diff: shows changes made to files
  • git rm filename: remove file from working directory
