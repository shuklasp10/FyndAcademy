Margin & Padding

margin give space around the element
padding give space inside element around the content

background-clip

background-clip specify how far the background should extend within an element
default value: border-box; bg extend behind border
padding-box; extend to inside edge of border
content-box; extend to edge of content only

Opacity

This will be not seen
Opacity specify the opacity of an element
Opacity value range from 0 to 1
0 value means transparent; not visible
1 value means opaque; fully visible
alpha value in hsla and rgba does not fade element content only apply to color, whereas opacity applies to both bg and content

background image

background-size property will determine size of bg image
background-repeat: no-repeat; will not repeat image in order to fill bg. By default it will repeat
background-attachment: set bg according to page
